# AI-Powered Resume Screening & Candidate Ranking System

An NLP-powered ATS system that parses resumes, extracts skills, compares candidates against job descriptions, and ranks applicants based on relevance.

## Features
- Resume Parsing
- NLP Preprocessing
- Skill Extraction
- TF-IDF Vectorization
- Cosine Similarity Scoring
- Candidate Ranking
- Streamlit Dashboard

## Tech Stack
Python, NLTK, SpaCy, Scikit-Learn, TF-IDF, Streamlit
